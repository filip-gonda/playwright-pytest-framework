# Playwright Pytest Test Automation Framework

This project is a test automation framework built with Python, Playwright, and Pytest.  
It is designed for UI test automation of https://automationexercise.com/ using a scalable and maintainable structure.

---

## Tech Stack

- Python
- Playwright
- Pytest

---

## Project Structure


tests/       -> test cases
pages/       -> POM classes
conftest.py  -> fixtures

---

## Framework Design (Page Object Model)

This framework follows the Page Object Model (POM) design pattern:

- Each web page is represented as a separate class
- UI actions and locators are separated from test logic
- Tests are structured to be clean, readable, and reusable

---

## Purpose of the Project

The goal of this project is to:

- Improve test automation skills using Playwright
- Practice scalable test framework design
- Apply Page Object Model architecture
- Gain experience with Python-based QA automation
